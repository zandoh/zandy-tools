# ZandyTools

A modular addon suite for World of Warcraft.

## Modules

- **Auto Role Check** - Automatically respond to role checks with preferred roles

## Usage

Type `/zt` to open configuration.

## Installation

Extract to `World of Warcraft/_retail_/Interface/AddOns/`

## Requirements

- World of Warcraft Retail (12.0.0+)
